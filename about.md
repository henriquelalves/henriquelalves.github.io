I'm Henrique Lacreta Alves - computer engineering student, storytelling lover and practical gamer. This is a site to host my projects and opinions. More about my curriculum and academic related projects on [Academic](/academic/).

The bottom of the site are quotes I gathered that made an impression on me, some way or another. It chooses a random line from [this gist file](https://gist.github.com/henriquelalves/4adf6d3ff0a04825ab30#file-quotes-txt) on each page refresh.

This site was created with [Jekyll](https://jekyllrb.com/), and the template is based on [Kiko](https://kiko.gfjaru.com/) - with increasingly more usability changes as I see fit.
